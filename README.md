# regex-cheat-sheet
Regex Cheat Sheet with the most needed stuff..


## Match everything including new lines
```javascript
// method 1
/[\S\s]/gmi

// method 2
/(?s:.)/gmi
```

## Replace all white spaces
```javascript
/\s\s+/gmi
```  


## Remove all special Characters
```javascript
/[^a-zA-Z0-9 ]/gmi
```  



## Match space after specific amount of characters
```javascript
/.{1,300}(\s|$)/gmi
```
